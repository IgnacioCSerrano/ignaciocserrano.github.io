Ejercicio 12

  1. Elemento <content>

    Esta propiedad se usa con los pseudoelementos ::before y ::after para generar contenido en un documento.

    Especificación W3C: Cascading Style Sheets Level 2 Revision 2 (CSS 2.2) Specification

  2. Elemento <border-image>

    Esta propiedad asigna una imagen como el borde de un elemento.

    Especificación W3C: CSS Backgrounds and Borders Module Level 3

  3. Elemento <border-shadow>

    Esta propiedad define la sombra de un elemento.

    Especificación W3C: CSS Backgrounds and Borders Module Level 3

  4. Elemento <resize>

    Esta propiedad permite especificar si un elemento puede cambiar de tamaño y a lo largo de qué ejes.

    Especificación W3C: CSS Basic User Interface Module Level 3 (CSS3 UI)

  5. Elemento <transition>

    Esta propiedad permite modificar el valor de una o varias propiedades de un elemento con fluidez, en una transición de duración determinada.

    Especificación W3C: CSS Transitions
