COMPARACIÓN ENTRE ELEMENTOS RSS Y ATOM EN PERIÓDICO HOY
RSS ATOM Significado
<rss version="2.0"> <feed xmlns="http://www.w3.org/2005/Atom"> Elemento que encapsula todo el contenido del canal (<channel> de RSS está subordinado a <rss>).
<channel>
<title> <title> Nombre del canal o de cada entrada individual.
<link rel="alternate"> <link rel="alternate"> URL del documento XML del canal o al sitio web HTML correspondiente a cada sección o entrada del mismo.
<item> <entry> Cada uno de los artículos individuales que componen el sitio web sindicado.
<guid> <id> Identificador único de cada artículo <item> o <entry>.
<pubDate> <updated> <pubDate> indica el momento en el que el artículo fue publicado (<published> en ATOM) mientras que <updated> indica el momento en que se realizó la última actualización al artículo (<lastBuildDate> en RSS).
<description> <summary> Texto descriptivo que resume el artículo correspondiente. Usa la sección <![CDATA[...]]> para evitar que el texto contenido sea interpretado como código HTML.
